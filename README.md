# StrideFlex - Made for India 🇮🇳

A modern, mobile-friendly single-page website for the Indian shoe brand "StrideFlex" built with HTML, CSS, and JavaScript. Designed to resemble popular Indian ecommerce brands like Campus Shoes, Sparx, and RedTape.

## 🚀 Features

### Design & Layout
- **Indian-Themed Design**: Red, black, and white color scheme inspired by Indian ecommerce brands
- **Modern Typography**: Poppins font family for bold, modern look
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: CSS transitions and JavaScript-powered animations
- **Indian Cultural Elements**: Tricolor effects, Hindi notifications, and Indian styling

### Sections Included
1. **Navigation Bar**: Fixed navigation with StrideFlex logo, Home, Products, Offers, and Contact
2. **Hero Banner**: Bold Indian-style tagline "Walk with Confidence – Made for India" with call-to-action
3. **Product Showcase**: 4 sneakers with product images, names, prices in ₹, and "Buy Now" buttons
4. **Why StrideFlex**: Features highlighting Lightweight Comfort, Affordable Pricing, and Designed for Indian Roads
5. **Promotional Offer**: "Flat 30% Off – This Week Only!" with countdown timer
6. **Newsletter Subscription**: Email signup form with validation
7. **Contact Form**: Name, Email, and Message fields with Indian contact information
8. **Footer**: Social media icons and "Made in India 🇮🇳" branding

### Interactive Features
- **Mobile Navigation**: Hamburger menu for mobile devices
- **Countdown Timer**: Real-time countdown for limited-time offers
- **Newsletter Form**: Email validation with Hindi success messages
- **Contact Form**: Complete contact form with validation
- **Buy Now Buttons**: Interactive product purchase simulation
- **WhatsApp Integration**: Floating WhatsApp button for customer support
- **Smooth Scrolling**: Navigation links smoothly scroll to sections
- **Hover Effects**: Interactive elements with hover animations
- **Hindi Notifications**: User feedback in Hindi language

## 🛠️ Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with Flexbox and Grid layouts
- **JavaScript (ES6+)**: Interactive functionality and animations
- **Font Awesome**: Icons for features and social media
- **Google Fonts**: Poppins font family for typography

## 📱 Responsive Design

The website is fully responsive and optimized for:
- **Desktop**: 1200px+ (full layout with side-by-side content)
- **Tablet**: 768px - 1199px (adjusted grid layouts)
- **Mobile**: < 768px (stacked layout with mobile navigation)

## 🚀 Getting Started

1. **Clone or Download**: Get the project files
2. **Open in Browser**: Simply open `index.html` in any modern web browser
3. **No Build Process**: This is a static website that runs directly in the browser

- Update product images, names, and descriptions in `index.html`
- Modify the countdown duration in `script.js`
- Change contact information and social media links
- Update Hindi notification messages

### Styling
- Adjust typography by changing Google Fonts import
